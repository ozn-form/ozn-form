はじめに ozn-form（オゾン・フォーム）

ozn-form では、コーディング済みのHTMLファイルに所定の記述することによって、ozn-formテンプレートとして扱うことができます。

ozn-form では、フォームを構成するファイルは公開ディレクトリの中でしたら自由な場所に配置できますので、既存のサイトなどにも制限なく組み込むことができます。

ファイル構成

構成サンプル

ozn-formで作成できるフォームのファイル構成サンプルを記載します。

確認画面なしシンプルフォーム（最小構成）

確認画面なしの最小構成のフォーム例です。

ozn-formを機能させる為に最低限必要なファイルになります。

ファイル構成
  •   (ドキュメントルート)
  • ┣   ozn-form ソースディレクトリ
  • ┣   フォーム設定ファイル
  • ┣   フォーム画面
  • ┗   送信完了画面

確認画面付きステップフォーム（最大構成）

フォーム画面が2画面と確認画面のあるSTEPフォームの構成例です。

こちらが最大構成となります。設計上はステップ数に特に上限はありません。

ファイル構成
  •   (ドキュメントルート)
  • ┣   ozn-form ソースディレクトリ
  • ┣   フォーム設定ファイル
  • ┣   フォーム STEP1 画面
  • ┣   フォーム STEP2 画面
  • ┣   確認画面 HTML
  • ┗   送信完了画面HTML

複数フォーム

ozn-formは設定ファイルに基づいて各処理を行うので、構成の異なる複数のフォームを設置することができます。

ファイル構成
  •   (ドキュメントルート)
  • ┣   ozn-form ソースディレクトリ
  • ┣   1つ目フォーム
  • ┃┣   フォーム設定ファイル
  • ┃┣   フォーム 画面
  • ┃┣   確認画面 HTML
  • ┃┗   送信完了画面HTML
  • ┗   2つ目フォーム
  •     ┣   フォーム設定ファイル
  •     ┣   フォーム 画面
  •     ┗   送信完了画面HTML
フォームHTML
<?php
  // 設定ファイルのパスを設定
  $config_path = 'ここに設定ファイルまでのパスを記載';

  // OznForm 実行ファイル読み込み
  require_once 'ここに実行ファイルまでのパスを記載';
?>

<!DOCTYPE html>
<html>
<head>
  <title>サンプルフォーム</title>

  <!-- ozn-form のデフォルトスタイルを読み込み -->
  <?php echo $ozn_form_styles; ?>

  <!-- ozn-form のデフォルトjsを読み込み -->
  <?php echo $ozn_form_javascript; ?>
</head>
<body>
 .....
</body>
</html>
