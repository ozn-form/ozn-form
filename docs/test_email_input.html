メールアドレス入力フィールド - 動作確認テスト

テストの目的

本ページは、メールアドレス入力フィールドの改善内容を視覚的に確認するためのテストページです。

  • type="text" inputmode="email" の動作確認
  • カスタムドメインサジェストの表示確認（簡易実装）
  • ブラウザのオートコンプリートとの競合確認

Test 1: 改善後の実装 推奨

実装内容: type="text" inputmode="email"

gmail.com
yahoo.co.jp
hotmail.com
outlook.com
<input type="text" inputmode="email" autocomplete="off" ...>

期待される動作:

  • モバイルデバイスでメール用キーボードが表示される
  • @記号の後にドメインサジェストが表示される（簡易実装）
  • ブラウザのオートコンプリートは抑制される（autocomplete="off"）

Test 2: type="email" の場合 非推奨

実装内容: type="email"

gmail.com
yahoo.co.jp
hotmail.com
outlook.com
<input type="email" autocomplete="off" ...>

問題点:

  • ブラウザによっては autocomplete="off" が無視される
  • ブラウザのオートコンプリートとカスタムサジェストが競合する可能性
  • 過去に入力したメールアドレスがブラウザから提示される

属性の比較表

type="email" email - ✅ 表示される ❌ 回避困難 ⚠️ 競合の可能性
従来 text - ❌ 表示されない ✅ 回避可能 ✅ 正常動作

操作方法

  1. 各入力フィールドに test@ と入力してください
  2. @記号の後に文字を入力すると、ドメインサジェストが表示されます（簡易実装）
  3. モバイルデバイスの場合、キーボードの種類を確認してください
  4. Test 2 では、過去に入力したメールアドレスがブラウザから表示される可能性があります
